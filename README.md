## Run Locally

Clone the project

```bash
  git clone https://github.com/ogodeverest/galaxy-creatures.git
```

Go to the project directory

```bash
  cd galaxy-creatures
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

**Client:** Three.js
